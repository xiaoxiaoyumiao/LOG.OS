在安装完成后，需要将 Node.js 的安装路径添加到环境变量 path 中。在 Windows 中可以通过 get-command 命令查看  npm 所在的目录（形如 .../nodejs），添加到 path 即可。同时为了能从命令行直接运行使用 npm 安装的 module，可以将 nodejs 目录下的 `node_modules`  和 `node_global` 也加入 path 变量中。
