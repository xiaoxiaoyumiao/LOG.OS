# Jupyter Notebook

## Installation

```bash
# use conda
conda install -c conda-forge notebook
# use pip
pip install notebook
```

## Basic Usage

```bash
# normal run to view with local browser
jupyter notebook

# run on remote server
# Replace <PORT> with your selected port number
jupyter notebook --no-browser --port=<PORT>
```
