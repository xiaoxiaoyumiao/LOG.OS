# Render Pipelines

Unity 默认使用 built-in render pipeline\(RP\)，但同时还有一些其他的 pipeline 可供选择，这些 pipeline 互不兼容，因此可使用的 shader 也互不兼容。切换一个已有项目的 render pipeline 时，场景可能会变成一片粉色。
